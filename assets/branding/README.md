# Migration Toolkit for Applications (MTA)

This extension provides support for the Migration Toolkit for Applications (MTA), helping you analyze and migrate applications to modern platforms.

## Features

- Application analysis and assessment
- Migration rule management
- Issue identification and remediation suggestions
- Integration with Red Hat's migration ecosystem

## Installation

1. Download the `.vsix` file from the [releases page](https://github.com/migtools/editor-extensions/releases)
2. In VS Code, run `Extensions: Install from VSIX...` from the command palette
3. Select the downloaded `.vsix` file

## Getting Started

1. Open your application workspace in VS Code
2. Open the Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P`)
3. Run the "MTA: New Analysis" command to start analyzing your application
